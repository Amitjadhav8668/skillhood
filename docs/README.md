# SkillHood - Location-Based Skill Exchange Platform

A full-stack web application for community-driven skill exchange without monetary transactions.

## 🚀 Quick Start

### Frontend
```bash
# Open index.html in browser or use:
npx http-server -p 3000
```

### Backend
```bash
cd backend
npm install
node seed.js        # Create test data
npm run dev         # Start server on port 5000
```

## 📁 Structure

```
skillhood/
├── backend/           # Node.js + Express + MongoDB API
├── *.html            # Frontend pages
├── styles.css        # UI styling
└── auth.js           # Authentication logic
```

## ✨ Features

- JWT Authentication
- Profile & Skill Management
- Location-based Matching
- Request System
- Real-time Chat
- Admin Panel
